### Introduce

This is web application that user can manage their products and other 


### Features

- Laravel Framework.
- React.js.
- Mysql DB.

### Instructions
Please change .env.example to .env
Create change db name in .env file
composer install
php artisan migrate
php artisan key:generate
php artisan serve
